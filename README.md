# DevSwings

**DevSwings** is a developer productivity tracker that helps you measure and improve your deep work habits. It combines a REST API backend with a terminal-based dashboard (TUI) so you can track coding sessions, git commits, streaks, and personal stats — all from the command line.

---

## Features

### 🔐 Authentication
- Register and log in with email/password
- GitHub OAuth login
- JWT-based access tokens with refresh and logout support

### 📁 Projects
- Create and manage multiple projects
- Set weekly hour goals per project
- Select the active project in the TUI to scope sessions and commits

### ⏱️ Work Sessions
- Start a session with your current mood and energy level (1–10)
- End a session with post-work energy, notes, blockers, and flow state (yes/no)
- View and manage all past sessions

### 📝 Commits
- Automatically sync local git commits to the API (auto-detected every minute in the TUI)
- Manually trigger a git sync with a single key press
- Batch import commits and link them to sessions and projects

### 🔥 Streaks
- Track your current consecutive-day coding streak
- Displayed live in the TUI sidebar

### 📊 Insights
- View aggregate stats: total sessions, hours, flow rate, commit count, average energy
- Historical insight records generated automatically by background workers

### 📬 Daily Digests
- Automated daily summaries generated by a Celery beat scheduler
- Retrieve digest history via the API

### 🖥️ Terminal Dashboard (TUI)
- Full-screen interactive dashboard built with [Textual](https://github.com/Textualize/textual)
- Live timer during active sessions
- Project list with click-to-select
- Keyboard shortcuts for every action

| Key | Action |
|-----|--------|
| `s` | Start session |
| `e` | End session |
| `m` | Sync git commits |
| `v` | View project report |
| `n` | Create new project |
| `r` | Refresh data |
| `l` | Login |
| `g` | Ghost Mode |
| `q` | Quit |

### 👻 Ghost Mode
- A minimal Rich-powered status bar that shows your active session timer without opening the full TUI
- Useful when you want to stay focused in the terminal

## Getting Started

### Prerequisites

- [Docker](https://docs.docker.com/get-docker/) and [Docker Compose](https://docs.docker.com/compose/)
- Python 3.11+ (for the CLI only)

### 1. Clone the repository

```bash
git clone https://github.com/sultaaaane/DevSwings.git
cd DevSwings
```

### 2. Configure environment variables

Copy the example file and fill in your values:

```bash
cp .env.example .env
```

| Variable | Description |
|----------|-------------|
| `POSTGRES_USER` | PostgreSQL username |
| `POSTGRES_PASSWORD` | PostgreSQL password |
| `POSTGRES_HOST` | Database host (`db` when using Docker) |
| `POSTGRES_DB` | Database name |
| `POSTGRES_PORT` | Database port (default `5432`) |
| `REDIS_HOST` | Redis host (`redis` when using Docker) |
| `REDIS_PORT` | Redis port (default `6379`) |
| `SECRET_KEY` | Secret key for JWT signing |
| `ACCESS_TOKEN_EXPIRE_MINUTES` | Token expiry in minutes (default `30`) |
| `GITHUB_CLIENT_ID` | GitHub OAuth App client ID *(optional)* |
| `GITHUB_CLIENT_SECRET` | GitHub OAuth App client secret *(optional)* |
| `ENVIRONMENT` | `DEV` or `PROD` |

### 3. Start the backend

```bash
docker compose up --build
```

This starts:
- **api** — FastAPI server on `http://localhost:8000`
- **worker** — Celery worker for background tasks
- **beat** — Celery beat scheduler for periodic tasks
- **db** — PostgreSQL on port `5434`
- **redis** — Redis on port `6380`

The interactive API docs are available at `http://localhost:8000/docs`.

## Running the CLI

Install the CLI dependencies in a virtual environment:

```bash
python -m venv .venv
source .venv/bin/activate        # Windows: .venv\Scripts\activate
pip install -r docker/requirements.txt
pip install textual rich          # TUI dependencies
```

### Launch the TUI dashboard

```bash
python cli/app.py
```

You will be prompted to log in on first launch. Your token is saved locally for future sessions.

### Launch Ghost Mode

Ghost Mode shows a minimal status bar in your terminal without the full dashboard. Start a session in the TUI first, then press `g` to exit into Ghost Mode, or run it directly:

```bash
python cli/ghost.py
```

Press `Ctrl+C` to exit Ghost Mode.

## API Overview

| Endpoint | Description |
|----------|-------------|
| `POST /auth/register` | Register a new user |
| `POST /auth/login` | Log in, receive JWT |
| `POST /auth/refresh` | Refresh access token |
| `POST /auth/logout` | Invalidate token |
| `GET /auth/github/login` | Start GitHub OAuth flow |
| `GET /auth/github/callback` | GitHub OAuth callback |
| `GET/POST /projects` | List or create projects |
| `GET/PATCH/DELETE /projects/{id}` | Read, update, or delete a project |
| `GET/POST /sessions` | List or create work sessions |
| `POST /sessions/{id}/end` | End an active session |
| `GET/POST /commits` | List or create commits |
| `POST /commits/batch` | Batch-import commits |
| `GET /streaks/me` | Get current streak |
| `GET /insights/me` | Get stats summary |
| `GET /insights/history` | Get insight history |
| `GET /digests` | List daily digests |
| `GET /health` | Health check |
